Detailed Description

A helper class that allows for simple "code block" implicit timer start/stops.

The following code: 

     {
         NvCPUTimerScope myScope(myTimer);
         // ... my block of timed code
     }
Is equivalent to: 
     {
         myTimer->start();
         // ... my block of timed code
         myTimer->stop();
     }
